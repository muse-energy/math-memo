# メモ

このページはVuepressで作っているのですが、数式を表示するために使っているプラグイン(vuepress-plugin-mathjax)があまり上手くいっていなくて、例えば

  • \begin とかが上手く使えない
    • 行列(pmatrix)が書けない
    • 複数行数式(align)が使えない
  • 一部の数式が意図せず別行立てで表示される場合がある

といった不具合がある気がしています。困りましたね…
