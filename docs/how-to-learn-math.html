# 数学の勉強法について

「こうすると良さそうかな？」ということを思いつくままに雑に書いてみます。 主に、「もう一度勉強したいけどどうすればいいんだろう？」って思っている大人の方向けの情報のつもりです。 人によって良い方法は変わりそうなので、ここに書かれていることにとらわれすぎず、自分がやりやすい方法を探ってみると良いような気がします。

数学との関わり方というか、どのようなスタンスで勉強していきたいか？というのも大事そう。

# 良さそうな勉強法

  • 本を読んだり、Youtubeなどの動画を見て学んだことを自分なりにまとめる。
    • 紙のノートにまとめる。
    • タブレットのノートアプリを使う。
    • パソコンでまとめる。
      • 自分はScrapboxを使っています。
      • このページもそれに近いかも
  • 他の人と一緒に勉強する。
    • 自分が学んだことを、他の人に説明してみる。
    • (可能であれば)詳しい人に質問できるようにしておく。
    • 一冊読みたい本を決めて、その本をみんなで読む(輪講、セミナー)。
  • 数学では、何かが成り立つ理由を、他の人にも伝わるように論理的に説明する活動が重要である。
    • 「証明」がこの活動に当たる。
  • 演習問題を解いてみる。
    • 簡単そうなものでもできるだけ丁寧に解答する。

# 大事そうなポイント

  • 論理に慣れる。
    • 「かつ」「または」「ならば」「～でない」「ある～」「～が存在する」「任意の～」「すべての～」などの意味に注意する。
  • 集合に慣れる。
